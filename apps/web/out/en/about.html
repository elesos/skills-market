About the project

Skills Market is a bilingual multi-page directory inspired by official skill ecosystems, with creators, repositories, skill detail pages, and search.

The frontend uses Next.js, while the backend is prepared for Cloudflare Workers with a future D1-backed catalog model.

Stack
  • Next.js 15 + TypeScript + Tailwind CSS
  • Cloudflare Workers API
  • Cloudflare Pages deployment target
  • D1-ready data layer

Built for fast exploration, clean browsing, and bilingual discovery.
