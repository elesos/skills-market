关于项目

Skills Market 是一个参考官方技能目录产品结构打造的多页面站点，支持中英文切换、创作者浏览、仓库详情、技能详情与搜索。

前端为 Next.js，后端预留 Cloudflare Workers API，并为后续 D1 数据化扩展做好结构准备。

Stack
  • Next.js 15 + TypeScript + Tailwind CSS
  • Cloudflare Workers API
  • Cloudflare Pages deployment target
  • D1-ready data layer

为高效浏览、清晰探索和双语发现而构建。
